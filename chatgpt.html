All projects below are open source and crafted with simplicity in mind.

List of Projects

  • move: A single bash script for moving files from source to destination. If file exists, it adds a numeric suffix.
  • move.js: Same functionality as above but in Node.js with more features.
  • move.zig: Same concept but written in Zig for performance.
  • ifc-cil: Finds today’s date in the International Fixed Calendar. Converts Gregorian to IFC format.
  • h.js: A tiny, zero-dependency DOM helper mimicking JSX and hyperscript using plain JavaScript.
  • HReact: A micro frontend framework built on top of h.js.
  • vent-wall: A minimal, judgment-free space to vent. Local storage only. No sign-up or tracking.
  • notes: A minimalist, full-page note-taking web app with offline support.
  • trash-for-termux: A bash script for Termux that replaces `rm`. It moves files to a trash folder instead of deleting immediately.
